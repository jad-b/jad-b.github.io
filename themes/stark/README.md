stark
=====
by [jad-b](https://github.com/jad-b)

A simple CSS theme, developed for [while-true-continue.com](https://while-true-continue.com).

## Layout
```
stark
 |   // Final resting place for compiled static assets
 +-- static
 |   +-- css/, js/, img/
 |   +-- robots.txt
 |
 |   // Pre-compiled assets
 +-- src
     +-- css/, js/
```
